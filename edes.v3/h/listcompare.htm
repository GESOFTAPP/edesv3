[ListCompare] ListaDeCampos | RanfoDeFechasIni, RanfoDeFechasFin[ | Lista de Nº de columnas [| nb,nb ] ]
DESCRIPCION
Listado comparando los datos entre un rango de fechas
PARAMETROS
ListaDeCampos: Nombres de campos separados por "," que se van a comparar.
RanfoDeFechasIni: Nombre del campo de fecha para el primer rango de fechas.
RanfoDeFechasFin: Nombre del campo de fecha para el segundo rango de fechas.
Lista de Nº de columnas: Lista de números de campos a mostrar, la primera columna es la posición cero, la primera vez que se utilice un número de columna será del primer rango de fechas y la siguiente vez que se use será del segundo rango de fechas.
nb,nb: Por defecto si en uno de los dos rangos no aparece ninguna entrada se pintará en la comparativa, si se quisiera que si no hay registro no aparezca se le pondrá "nb" ( NoBlanck ), el primer parámetro es para el rango inicial y el segundo para el final.
