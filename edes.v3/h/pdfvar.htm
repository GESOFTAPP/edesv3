PDFVar
SINTAXIS
[PDFVar] PDFVar = Value
...
DESCRIPCION
Nos permite modificar variables de configurac?on del PDF, es opcional el "$" delante de la variable as? como el ";".
Existe una variable del sistema "$_PDFLABELHIDDEN" para poder poner el label a campos que no la tengan o a campos de las condiciones que pueden no generarse en el listado ej: $_PDFLABELHIDDEN['NomCampo'] = 'label';
Si este 'label' tiene el valor especial "NotToShow" si el campo est? condicionado no se mostrará en la lista de las condiciones.
La variable $_PDFVALUELHIDDEN['Valor'] = Etiqueta te muestra un valor diferente en la información de condiciones.
El prefijo "PDF_" es opcional en la definición de la etiqueta, es obligatorio si se define la variable dentro de otra etiqueta.

$PDF_Condition: Texto a mostrar en la condición del PDF, esta variable puede ser una cadena o una matriz.
