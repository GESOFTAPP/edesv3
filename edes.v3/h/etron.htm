eTron
SINTAXIS
eTron( $txt [, $ConDelimitador=false [, $InicializarTron=false/SQL ] ]  )
eTron( Array )
DESCRIPCION
Es una función de depuración y rastreo, solo se debe utilizar en tiempo de desarrollo. Graba en un fichero de texto personal el contenido de la variable "$txt", este fichero es visible desde el editor "gsEdit" desde la opción "TRON leer".
Si el parámetro "$InicializarTron" le asignamos la clave "sql" el eTron() se verá por la opción de "Ver SQL".
Cuando esta función se ejecuta e un proceso Background la información se graba en el fichero "/_tmp/err/__tron.[$_User]" y si $txt=NULL limpiará el fichero.
El parámetro "$txt" puede ser una cadena o una matriz.
