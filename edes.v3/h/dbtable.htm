DBTable
SINTAXIS
[DBTable] Table1 [, Table2] ... [, TableN] [ [ | SufijoISubList ] | TablaInfo1,,,, ]
DESCRIPCION
Nombre de la tabla de la base de datos a utilizar.
Se puede poner más de una tabla, en ese caso el motor las utilizará para el listado y podría nombrarse con un alias (Factura as F), debiéndose utilizar también en los campos.
En un archivo de definición ".LST" (listado). Al tener más de una tabla, en los campos hay que definir la tabla a que pertenecen.
PARAMETROS
Table1: Nombre de la tabla de la base de datos que estamos utilizando.
El tercer parámetro opcional es una lista de tablas a ver su estructura desde el editor pulsando F2. El editor tiene una opción para ver la estructura de las tablas definidas en la etiqueta [DBTable] si tu en alguna parte de PHP usas mas tablas y quieres poder consultar la estructura SQL desde el editor las añades en la etiqueta.
EJEMPLO
[DBTable] clientes
