Descripción
E-DES es un sistema de desarrollo rápido para generar intranets corporativas, aplicaciones de gestión.

Con E-DES nace un nuevo concepto de desarrollo: "programación orientada a intranets", en efecto, el sistema ha sido diseñado exclusivamente para tal cometido, dadas sus características es uno de los modelos más vanguardistas y potentes del panorama actual en este ámbito.

El sistema se compone básicamente de dos elementos, el motor de desarrollo y el motor de producción, estos elementos están enlazados directamente como veremos más adelante.

El motor de desarrollo es la herramienta desde la que se programan todos los procedimientos que componen la Intranet, esta herramienta se maneja directamente desde el explorador de Internet, no necesita ningún tipo de instalación en el terminal del programador, por lo que es factible realizar modificaciones en cualquier terminal conectado a la Intranet desde cualquier punto geográfico mundial; podríamos estar programando o actualizando procedimientos en la propia ubicación del Cliente final, esto implica un ahorro de tiempo considerable si pensamos en la fase de implantación y en el mantenimiento posterior en la fase de producción. A diferencia de otros entornos no tenemos que preocuparnos por el nombre ni la ubicación de los archivos de definición, sólo cuando necesitemos ejecutar algún procedimiento, desde una función Javascript por ejemplo, deberemos conocer su nombre y ubicación, pero esta es una tarea trivial puesto que el nombre nos lo proporciona el propio sistema y la ubicación la sabremos de antemano desde la realización del análisis.

El motor de producción es el corazón del sistema, utiliza la tecnología "data driver" y se encarga de abastecer de información a los clientes, es el nexo de unión entre el usuario y la base de datos. La efectividad de este motor le permite abastecer de información a más de 1.000 usuarios utilizando la Intranet continuamente. Dispone de un completo sistema de seguridad tanto de acceso a la propia Intranet como de gestión de menús independientes por tipos de usuario.

Los usuarios finales usan la Intranet por medio del navegador de Internet, no necesitan tener instalado ningún programa adicional, esto nos indica que los ordenadores de los que deben disponer no deben tener unos requisitos elevados.

E-DES es un motor de aplicaciones de gestión, es similar a los motores de base de datos añadiendoles un interface.
Comparativa entre los motores de base de datos (DB) y el motor eDes:
DB  eDes 
Se crea un diccionario de datos  Se crea un proyecto
Se crean las estructuras de las tablas Se crean los ficheros de definición
Se ejecutan operaciones sobre las tablas: insert, update, delete  y select. Se ejcutan los ficheros de definición DF en distintos modo: alta, baja, consulta,  modificación y listado
El motor de base de datos hace todas las operaciones sobre  cualquier tabla El motor eDes hace todas las operaciones sobre cualquier fichero de definición y en cualquier modo
