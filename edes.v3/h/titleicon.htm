[TitleIcon] Mode | IconoHtml [ | NomDF,... / else ]
[TitleIcon] Mode | HPCV   [ [ [ | NomDF,... / else ] | FileHelp ]  | ClaveHelp ]
DESCRIPCION
A la derecha del titulo de la ficha se puede poner uno o mas iconos de acceso rápido, así como en los listados a la izquierda de los lTools.
PARAMETROS
Mode  Modo 
IconoHtml / HPCV   Hay dos formas:
1.- Poner el html de un icono
2.- Las iniciales (H)tml, (P)DF, (C)HM (V)ideo, una varias o un asterisco para indicar los cuatro tipos de ficheros
NomDF  Condiciona la etiqueta si viene de tal DF o "else" si no se ha inicializado
FileHelp  Cuando se define HPCV tomará ese nombre de fichero de ayuda:
- si se deja en blanco será el nombre del script y el del modo.
- se puede llamar a una función. 
- si el nombre de la ayuda lleva extensión será un fichero externo en /help/tip/

Para definir la ayuda dentro del edf se utilizan las etiquetas [HelpHTML] o [HelpMarkdown]
ClaveHelp  Cuando se usa el formato chm se puede saltar a una zona en particular con este par?metro.
EJEMPLO
[TitleIcon] * | < img title="..." src="g/historico.png" / >
