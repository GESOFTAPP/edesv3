MsgSubmit
SINTAXIS
[MsgSubmit] Mode | Message/FunctionJS [ [ [ | CondicionJS ]  | AnchoPx [ | TextAlign ]
DESCRIPCION
Con esta etiqueta podemos mostrar al usuario un mensaje de confirmación antes de hacer el submit. En el lado del cliente se puede modificar el contenido del mensaje mediante la variable javascript "_MsgSubmit". En caso de que el mensaje de confirmación se muestre en función de unas condiciones se especificará con el tercer parámetro "CondicionJS".
PARAMETROS
 Mode Modo en el que se muestra el mensaje
 Message/FunctionJS Mensage a mostrar en la confirmación,  si DefinitionFile empieza por ">" será el nombre de un fichero externo y si la extensión de este fichero es "LNG" lo sustituirá por las siglas del lenguaje del usuario.
 CondicionJS Condición javascript para mostrar el mensaje
EJEMPLO
[MsgSubmit] a,mR | ¿ Confirmar .... ?
[MsgSubmit] a,mR | ¿ Confirmar .... ? | | 450 | justify
[MsgSubmit] a,mR | ¿ Confirmar .... ? | uBorrar==true
[MsgSubmit] a,mR | ¿ Confirmar .... ? | MostrarConfirmacion()
[MsgSubmit] a,mR | uConfirmacion()
