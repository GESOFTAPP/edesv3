Setup
SINTAXIS
[Setup] Mode | listVar ...
DESCRIPCION
Modifica variables de configuración
PARAMETROS
El parámetro "listVar" es una lista de variables con su valor, en caso de que el valor esa "true" se puede omitir.
Actualmente solo está activa la variable "activateFlush" que desactiva el buffer de salida.
EJEMPLO
[Setup] l | activateFlush
En este ejemplo se activa el flush en los listados, por defecto se guarda la salida en el buffer y se manda al terminar de generar la página, esta funcionalidad nos permite por ejemplo en cualquier punto de la generación limpiar el buffer y generar otro contenido, esto consume mas memoria, cuando tienes que generar listados muy grandes sin paginación activas esta variable permitiendo generarlos.
