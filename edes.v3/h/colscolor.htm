ColsColor
SINTAXIS
[ColsColor] Colors, Colors, ...
 Sintaxis alternativa
[ColsColor] NomCampo=Colors, ...
DESCRIPCION
Lista de colores de las columnas en un listado.
Si empieza por una almuadilla "#" indicamos que es un COLOR, si no es el nombre de una clase que deberá estár definida dentro del archivo de hojas de estilo CSS y podemos poner "+", "-" y "+-" para indicar que los valores numéricos mayores de 0 se ponga la clase "numPositive" y si es negativo "numNegative" (rojo y verde respectivamente como valor por defecto)
PARAMETROS
 
EJEMPLO
[ColsColor] #cccccc, NomClaseTotales, +-
 
EJEMPLO
[ColsColor] dni=#dddddd
