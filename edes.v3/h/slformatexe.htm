slFormatExe
SINTAXIS
{slFormatExe} NameFunction
DESCRIPCION
 Por cada fila de la sublista te pasa un array hash con la siguiente información:
[_vF]
     [0] =>
     [1] =>
     [2] =>
     ...
[_CellsStyle]
     [0] =>
     [1] =>
     [2] => color:red
     ...
[_CellsClass]
     [0] =>
     [1] =>
     [2] =>
     ...
[nFila] => 0
[_RowStyle] =>
[_RowClass] =>
[_RowDisabled] =>

Si quieres poder modificar los datos tiene que preceder al parámetro con el símbolo "&"
