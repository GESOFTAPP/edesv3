eAddMenuOption
SINTAXIS
eAddMenuOption( $Label, $HR='', $Icon='', $Title='', $Activo=true  )
DESCRIPCION
En el arbol de opciones se puede definir una opción en la que se podrá definir opciones dinámicas en la carga del Desktop, para ello se definirá la URL de la opción como "PHP:" mas la función a ejecutar y desde dentro de la función de usuario se podrán insertar nuevas opciones.
PARAMETROS
EJEMPLO
-> En la URL de una opción pondremos:
PHP:uAltaMenus()

function uAltaMenus(){
eAddMenuOption( 'Opcion - A', 'uURL("edes.php?Fa:fpago.edf", "CAJA1")' );
eAddMenuOption( 'Opcion - C', 'uURL("edes.php?Fc:fpago.edf", "CAJA2")' );
eAddMenuOption( '-' );
eAddMenuOption( 'Opcion - M', 'uURL("edes.php?Fm:fpago.edf", "CAJA3")' );
}
