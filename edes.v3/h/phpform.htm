PHPForm
SINTAXIS
[PHPForm] Mode [[ | NomDF/else,... ] | UNIQUE/Condition ]
...
DESCRIPCION
Incluir codigo PHP para manipular la matriz que genera el formulario.
El código se ejecutará justo antes de generarse el formulario, lo que hace de esta etiqueta una herramienta importante a la hora de manipular un formulario antes de que sea generado. En las Multifichas se hará una llamada a la función por cada solapa.  Al label [PHPorm] se le pasan cuatro parametros que són:  &$_Form, $Opcion, $Fichero, $nHoja y &$_vF.
El contenido de una variable virtual se puede definir aquí con "$_vF['_NomVariable'] = '...Valor...';".
La variable $_Form se maneja de la forma $_Form[...NmField...][NumCol] y el NumCol se puede usar los siguientes define:
0: _LABEL
1: _FIELD
2: _EDITION
3: _CONTROL
4: _SIZE
5: _WIDTH
6: _MODE
7: _DEFAULT
8: _CONDITION
9: _MESSAGE
PARAMETROS
El parámetro "UNIQUE" es para incluir esta zona de código una sola vez en caso de tener funciones declaradas y halla conflicto con los modos.
EJEMPLO

[PHPForm] a,mR
if( ePermission('BorrarFile') ) $_Form['fichero'][_MODE] = $_Form['fichero'][_MODE].'d';
Si el usuario tiene permiso para borrar el fichero le añadirá a la columna Mode del parámetro "d" que indica que se pueda borrar.
