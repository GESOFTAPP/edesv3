¿Para quién esta diseñado E-DES?
E-Des es un producto especialmente diseñado para la Intranet de Gestión. E-des posibilita la migración al entorno de Web, tan necesaria hoy día, de las aplicaciones creadas sobre plataformas cliente-servidor que obliga a los usuarios a disponer de copias de programas instaladas en sus pc. E-des esta indicado principalmente para integradores de sistemas y fabricantes de software vertical, CRM,etc.

Esta indicado también para los departamentos de desarrollo de las empresas que pone a su disposición una herramienta potente para el rediseño de sus programas de gestión interna, convirtiendolos en intranets corporativas de alta distribución geografica y coste minimo.

También permite a los Webmaster utilizar herramientas diseñadas con e-Des para la gestión y control del parque de servidores. A los programadores de entorno personal les posibilita realizar programas sencillos para pc basados en el navegador.

Los diseñadores de base de datos tienen con e-Des una gestión centralizada y grafica de sus modelos de datos así como facilidad para mantener una gestion de cambios de los datos mas fiable y comoda de tratar.
