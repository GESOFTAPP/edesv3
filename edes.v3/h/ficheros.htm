Tipos de archivos:
Para realizar su cometido, el sistema se nutre de distintos tipos de archivos, la extensión que adopta cada uno de ellos depende de su funcionalidad.

Tipos de archivo gestionados directamente por el sistema.
 edf  Definición de fichas.
 gdf  Definición de grupos de fichas.
 ldf  Definición de listados de busqueda, parámetro 7 de la etiqueta [FIELDS]) contiene una 'b' (búsqueda por listado). 
 fdf  Definición para los campos de tipo 'SS' (subselects) y cuando el modo parámetro 7 de la etiqueta [FIELDS]) contiene una 'B' (búsqueda por ficha). 
 idf  Es un fichero de definición del que se hace include desde otro fichero mediante la directiva "#Include()". 
 zdf  Es un fichero de definición que se graba encriptado. 

Tipos de archivo NO gestionados directamente por el sistema   

.sel Código PHP para los campos del tipo 'SX' (select especial).
.ini Archivo de configuración por defecto.
.css Hojas de estilo en cascada.
.gs  Archivo que contiene código PHP.
.php Archivo que contiene código PHP.
.inc Contiene código Javascript/PHP para realizar inclusiones cuando lo precisemos.
.js  Contiene código Javascript.
.str Structura SQL de una tabla
.unl Descarga de los datos de una tabla.
.imp  Archivo para la importación de datos (no es gestionado directamente por el sistema).
.html Código HTML.
.txt  Archivo de texto plano.

Desktop.ini
Por defecto el árbol de opciones se coge del subdirectorio tree, pero esto es configurable mediante la variable $_TreeType (no disponible en los desktop tipo 0 y 1) que puede tener los valores:
"" Vacio o no estar definida es el comportamiento normal, se toma el fichero del directorio "tree".
"T" de Table para coger las opciones de la tabla de opciones directamente de la tabla
"A" de Array toma las opciones de la variable $_TreeArray, cada indice de este array se contruye de la siguiente forma:
- Nº de tabuladores en función de su indendación
- Label
- "|" delimitador vertical
- Orden a ejecutar
- "~" Delimitador
- Nº de la opción
- "~" Delimitador
- Title si lo tubiera
