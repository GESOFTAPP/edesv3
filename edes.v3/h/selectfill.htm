SelectFill
SINTAXIS
[SelectFill] Campo1,Campos2,...  /  mode / *
DESCRIPCION
Por defecto los campos select no editables no se rellenan, para que se rellenen se utiliza esta etiqueta.
PARAMETROS
El parámetro puede ser una lista de nombres de campo o una lista de Modos o el "*" para indicar todos los campos select.
