H
SINTAXIS
{H} RefName [ | NoTD ]
DESCRIPCION
Es una sub-etiqueta de [Fields].
Añadir código HTML en una fila de la rejilla del formulario.
Dentro de la etiqueta [Fields], en la definición de un campo, "{H}" seguido de un nombre de referencia (por ejemplo "myRef"). Después de la etiqueta [Fields] volvemos a insertar "[H]" seguido del mismo nombre de referencia y en las siguientes líneas el código HTML que necesitemos. Con éste método podemos instroducir código HTML a nuestro antojo en cualquier fila de la rejilla del formulario.
PARAMETROS
RefName: Nombre identificativo.
Si la constante "NoTD" no se pone te dejará una única celda con el cospan que corresponda para que metas lo que quieras, pero si le pones "NoTD" tu tienes que definir todos los TD de la fila.
EJEMPLO

[Fields]
   Client Code | cd_cli | ... 
   {H} myRef
...

[H] myRef
<table border=1>
<tr>
<td height="10" width="600" cellpadding="1" cellspacing="1" align=center>
<strong>
<font color="#ff0f00" size="3"> AVISO </font>
</strong>
</td>
</tr>
</table>
