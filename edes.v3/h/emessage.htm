eMessage
SINTAXIS
eMessage( $Mensaje, $Accion, $mlsg, $exe )
DESCRIPCION
Se mostrará la página con un único elemento un mensaje central.
La variable "$_EMESSAGE" cambiará el texto a mostrar.
PARAMETROS
$Mensaje: Mensaje a mostrar.
Valores de $Acción
E Error 
H Nuevo HTML
S Stop a la ejecucion y si no es 'IWORK' se cerrará
L Error en el login
V Dejar el mensaje Visible
C   Cerrar ventana inmediatemente
$mlsg: Milisegundos de espera del mensaje, si el parámetro es -1 el mensaje se quedará visible hasta hacer click en él.
$exe: Es código JavaScript:
     - Si empieza por ">" será el nombre de un fichero javascript
     - Si empieza por "<" se ejecutará en el header
     - Si no se ejecutará cuando se pulse click o termine el tiempo "$mlsg".
EJEMPLO
eMessage( 'PDF Descargado', 'HS' );
