TipForm
SINTAXIS
[TipForm] Mode [ | L / F / * / # ] | Field [, Field... ] [ | Text [ | Width [ | Sg ] ] ]
...
DESCRIPCION
Genera un TITLE en un determinado campo, la ayuda es visible al ponerte con el ratón encima del label y/o del campo, también cuando el foco se pone en un campo que tiene ayuda se mostrará durante 2 segundos independientemente de su tiempo de exposición y pulsado F1 se mostrará la yuda sin tiempo de caducidad.
También se puede visualizar la ayuda desde un icono mediante la función "eTipForm(NomCampo)"
PARAMETROS
Field: Nombre de un campo existente en la etiqueta [Fields].
 Mode  Modo
 L / F / *
    I
   E
   #		 Este campo es opcional si no se define el Tip irá en el Field, sino será:
L: En el Label
F: En el Field
*: En los dos o dejarlo en blanco
I: En icono a la derecha del campo
E: En icono a la derecha del campo con el texto en un fichero externo
#: Indica que en la posición del Field habrá el valor del "id" de un objeto al que se le asigna la ayuda.
 Field Nombre del campo o lista de campos separados por coma.
TITLE: Para el titulo, el los GDF se define en el GDF.
BUTTON: Para el botón en submit.
AddButton[nº]: Los botones añadidos con [AddButton]
 Text Texto a mostrar, si hay un "#" se sustituirá por el valor del campo.
Funcion()
Si empieza por ">" le seguirá el nombre de un fichero html
Si está vacio el contenido irá como multilinea.
El "Texto" tanto en linea como en multilinia admite variables {$variable} y/o funciones o
 Width  Ancho de la ayuda
 Sq  Nº de segundos a mostrar la ayuda, si se omite será hasta que se quite el cursor
