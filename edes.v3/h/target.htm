SINTAXIS
[Target] format:maxRecords, ...
DESCRIPCION
Selecciona el destino de un listado en función del número de registros.
PARAMETROS
Es una lista de formatos de exportación con el máximo número de registros a enviar. Si el primer destino no tiene definido un número de registros será el destino de salida cuando el número de registros es inferior a la siguiente definición y cuando es el último significa que cuando sea mayor que el penúltimo.
Se puede indicar mas de un formato con el mismo número de registros separandolos por "/".
EJEMPLO
[Target] html, pdf:2000, xml/xls:65000, csv
