Objetos

Ficha: Llamamos genéricamente "ficha" a la ventana simple que está compuesta de un solo archivo EDF.
Una ficha puede desempeñar distintos propósitos, altas, modificaciones, bajas, filtro de datos para selección de un registro, etc. Las fichas también pueden manejar en modo consulta y modificación ficheros ".ini" de forma transparente, si existe un fichero con el mismo nombre pero con la extensión ".js" lo cargará como si fuera la etiqueta [JSEnd].

MultiFicha: Las multifichas son agrupaciones de fichas. Cada una de las fichas se muestra en una solapa diferente y normalmente están relacionadas entre sí, de esta forma podemos tener mucha información en pantalla, perfectamente organizada y además sin tener que hacer ningún scroll de la ventana.

Listado: Los listados pueden tener dos propósitos, bien visualizar simplemente los datos en forma de lista o bien seleccionar un dato para llevarlo a otra ventana.
Tipos de listado:
- Normal
- Agrupaciones
- Comparativas
- De rejilla
- Listado editable
- Listado de ordenación
Exportación en todos los listados a:
- PDF ( b/n y color )
- XLS
- XML
- CSV
- TXT
Gráficas  compatibles con Flash de la casa  PHP/SWF Charts (http://www.maani.us/charts/index.php). 

Etiquetas: En formato PDF ( fichas )
