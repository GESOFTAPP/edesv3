NoEditFilled
SINTAXIS
[NoEditFilled] Mode [ [ | session ]  | ListaDeCampos ]
DESCRIPCION
Evita la edición de los campos que esten rellenos con un valor por defecto cuando este tenga contenido. Puedes tener usuarios con distinto tipo de campos a editar.
PARAMETROS
Mode  Modo de ejecución
Session  Si se pone el parámetro "session" evitará la edición sólo de las variables de sesión si están rellenas. 
ListaDeCampos  Sólo desabilitará la edición a estos campos que estén rellenos
EJEMPLO
[NoEditFilled] a
No dejará editar ningún campo que esté relleno
EJEMPLO
[NoEditFilled] a | session
No dejara editar ningún campo que esté relleno mediante variables de sesión si está rellenos, si hay campos rellenos pero no son de variables de sesión si los dejará editar.
EJEMPLO
[NoEditFilled] a | | campo1, campo2
No dejará editar los campos que estén rellenos de la lista de campos
